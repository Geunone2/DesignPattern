# 사용할 패턴 : 팩토리, 어댑터, 데코레이터

### 🏭 팩토리 패턴
각 회사 별(Geunone, Geuntwo, Geunthr) 네비게이션마다 추천하는 경로를 보여주는 기능을 출력하도록 작성<br>
Ex) Geunone => 고속도로 / Geuntwo => 일반국도 / GeunThr => 지방도 etc,,,
